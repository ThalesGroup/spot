# Contributing guidelines

This repository is part of the following collaborative project:
"Towards Operational Quantum Computing for Earth Observation" (QR4EO) 

## How to become a contributor
If you wish to contribute to SPOT, you can send an email to
michel.nowak@thalesgroup.com and oss@thalesgroup.com in parallel.

### License
The [LICENSE](/LICENSE.md) is Apache 2.0

### Testing
Before commiting a development, always test one example which will test the
feature. If it's not possible, try writing an example that will do.

#### Running unit tests
We do not have unit tests yet.

### Issues management
If you find an issue, please use the Github page dedicated to issues.
We will treat the issue as soon as possible
